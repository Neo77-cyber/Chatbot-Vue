# Todo App and Chatbot with Kommunicate

This project is a Todo App integrated with a Chatbot powered by Kommunicate, built using Vue.js.

## Features

- Todo App:
  - Create, update, and delete tasks
  - Mark tasks as completed
 

- Chatbot:
  - Integrated Kommunicate chat widget for real-time communication
  - AI-powered chatbot to provide automated responses and assistance

## Installation

1. Clone the repository:
2. Navigate to the project directory:
3. Install the dependencies:
4. Set up the environment variables:
- Create a `.env` file in the root directory
- Add the following variables to the `.env` file:
  ```
  VUE_APP_KOMMUNICATE_APP_ID=your_kommunicate_app_id
  ```

## Configuration

To configure the Chatbot integration with Kommunicate:

1. Sign up for an account on [Kommunicate](https://www.kommunicate.io/).

2. Create a new app and note down the App ID.

3. Update the `VUE_APP_KOMMUNICATE_APP_ID` variable in the `.env` file with your Kommunicate App ID.

## Usage

1. Start the development server:

2. Open the app in your browser:

3. Use the Todo App to manage your tasks and the Chatbot to interact with the automated assistant.
